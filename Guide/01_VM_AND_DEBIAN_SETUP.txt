Debian Setup

	1) Install sudo: 
	
	   a) Switch to the root user:  su -

	   b) Update and upgrade the installed packages:  apt-get update && apt-get upgrade -y

	   c) Installs the sudo package, allowing users to execute commands with admnistrative privileges:  
	      apt-get install sudo -y

    2) Add User to the Sudo Group: 
	
	   a) Add user to the sudo group:  adduser rmedeiro sudo

	      Grants admnistrative privileges to the user through Debian sudo group.
	
	   b) Check that the user belongs to the sudo group:   getent group sudo

	   c) Apply the change rebooting the system:  sudo reboot

	3) Update system:  sudo apt update && sudo apt upgrade -y

	4) Install Vim: sudo apt install -y vim

	5) Go to sudoers file: sudo visudo
	   Edit the line below the root line adding username and the privileges: rmedeiro ALL=(ALL:ALL) ALL

	6) Install OpenSSH Server: 
	
	   a) Install OpenSSH server:  sudo apt install -y openssh-server

	      Allows remote connections to the virtual machine using SSH

	   b) Enable SSH at boot:  sudo systemctl enable ssh

	   c) Start the SSH service:  sudo systemctl start ssh

	   d) Check that SSH is running:  sudo systemctl status ssh

	      Expected status: active (running).
     
	7) Change SSH port 22 to 4242: 
	
	   a) Open the SSH configuration file:  sudo vim /etc/ssh/sshd_config

	   b) Find: #Port 22 and replace with Port 4242. Save and exit.

	      Changes the SSH listenning port 22 to 4242.

	   c) Restart SSH:  sudo systemctl restart ssh

	   d) Check:  sudo systemctl status ssh (check)

	8) Configure VS COde Remote SSH:  

	   a) Install the Remote - SSH extension in VS Code.

       b) Connect to the VM: Open VS Code and press Ctrl + Shift + P .

	   c) Select: Remote-SSH: Connect to Host... and choose Add New SSH Host...

	   d) Enter: ssh -p 4242 rmedeiro@<VM_IP>  (Example: ssh -p 4242 rmedeiro@192.168.1.120)

          To obtain the VM IP: hostname -I .

	   e) Select the SSH config file, connect to the host and enter the VM password when prompted.
